# @expo98/ux-context-capture

Modernized TypeScript transformation of the legacy `captureUxContext` evidence assembler from `legacy/expo98/dist/expo-ios.mjs`.

This package composes project metadata, iOS device metadata, Metro/runtime state, app container info, screenshots, image analysis, route context, simulator hierarchy, and filtered log evidence into the UX review context payload.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:3545-3656`.
