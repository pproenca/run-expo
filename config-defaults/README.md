# @expo98/config-defaults

Modernized TypeScript transformation of the expo98 central configuration defaults and state path rules.

This package captures source-backed defaults for CLI identity, output limits, Metro port, parse globals, state roots, run-record directories, artifact directories, environment keys, and the `ExpoIosConfig` default shape.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/src/contracts/config.ts`, `legacy/expo98/dist/expo-ios.mjs:12-18`, `legacy/expo98/dist/expo-ios.mjs:12012-12014`, `legacy/expo98/dist/expo-ios.mjs:12780-12799`, and `legacy/expo98/dist/expo-ios.mjs:12996-13004`.
