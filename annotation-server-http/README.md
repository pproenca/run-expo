# @expo98/annotation-server-http

Modernized TypeScript transformation of the legacy annotation server request handling from `legacy/expo98/dist/expo-ios.mjs`.

This package routes annotation-server HTTP requests as data: static annotation files, `POST /annotations` persistence, JSON response formatting, request body limits, and startup payload metadata.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:3758-3794` and `legacy/expo98/dist/expo-ios.mjs:4152-4182`.
