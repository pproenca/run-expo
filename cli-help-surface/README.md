# @expo98/cli-help-surface

Modernized TypeScript transformation of the legacy expo98 CLI help and usage surface.

This package exposes the help text as structured command groups, global flags, examples, and a `printHelp` boundary that can be injected into CLI dispatch.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:13093-13258`, with supporting contract context from `legacy/expo98/README.md` and `legacy/expo98/SPEC.md`.
