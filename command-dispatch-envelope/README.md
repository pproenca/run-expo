# @expo98/command-dispatch-envelope

Modernized TypeScript transformation of the legacy expo98 CLI dispatch and output envelope behavior.

This package exposes the runtime command alias table, parsed-command dispatcher, tool invocation wrapper, JSON/plain/default output formatting, output bounding, error envelopes, and secret redaction used around legacy command execution.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:12071-12213` and `legacy/expo98/dist/expo-ios.mjs:12918-12994`.
