# @expo98/review-sidecar-state

Modernized TypeScript transformation of the durable state helpers behind the legacy review overlay and annotation sidecars.

This package covers endpoint path normalization, overlay event file lifecycle, event append semantics, component-stack symbolication enrichment, annotation payload persistence, and JSON response formatting.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:4044-4182`.
