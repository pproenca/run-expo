## Deliberate Deviations

| Deviation | Rationale |
| --- | --- |
| Metro target discovery and Hermes evaluation are injected dependencies. | This matches the existing modernization pattern for runtime adapters and keeps tests deterministic without a live Metro server. |
| `stateRoot` and `session` are accepted in the context type but not read. | The legacy helper received them through the snapshot command context but only used `filters`; preserving the context shape keeps call sites compatible. |
