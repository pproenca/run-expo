# @expo98/tool-schema-builders

Modernized TypeScript transformation of the legacy expo98 tool schema and text-result builders.

This package exposes the schema helpers used by the legacy tool catalog plus the `toolText`, `toolJson`, and `unwrapToolJson` helpers used around command handler payloads.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:772-813`.
