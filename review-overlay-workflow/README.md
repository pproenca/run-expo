# @expo98/review-overlay-workflow

Modernized TypeScript transformation of the legacy `reviewOverlay` and `scaffoldReviewOverlay` workflow from `legacy/expo98/dist/expo-ios.mjs`.

This package prepares review overlay event state, reads and clears events through injected state adapters, returns detached server launch metadata, and scaffolds a React Native `CodexReviewOverlay` component plus integration instructions.

## Commands

```bash
npm test
```

The tests are characterization tests derived from `legacy/expo98/dist/expo-ios.mjs:3797-3901`.
