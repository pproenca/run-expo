# @expo98/png-screenshot-analysis

Modernized PNG screenshot analysis helpers for expo98 UX context evidence.

This package covers:

- PNG dimension reads
- PNG decoding for 8-bit grayscale, RGB, grayscale-alpha, and RGBA screenshots
- scanline unfiltering
- sampled color palette and luminance summaries
- coarse visual composition metrics
